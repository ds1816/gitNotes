git 是分布式版本控制软件

从本地工作目录和版本库中间有一个  暂存区，
本地工作目录 到暂存区 命令 git add  暂存区到版本库 命令 git commit 

在互相推送版本代码时 git push  反之（拉取远程代码更新本地代码版本） git pull


本地版本库有多个（分支1,2,3....）,分支默认主分支为master，即暂存区到版本库（分支）默认commit到master上
